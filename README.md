# Cloud Motion Tracking for Earth Observation

This project explores computer vision techniques for detecting and tracking cloud movement in Earth Observation imagery.

Cloud motion analysis is important for atmospheric research, weather prediction, and satellite-based wind estimation.

## Motivation

Tracking cloud patterns over time allows estimation of atmospheric dynamics such as wind direction and speed. Lightweight computer vision models can enable real-time analysis of satellite imagery.

## Objectives

- Detect cloud regions in satellite images
- Track motion between consecutive frames
- Estimate cloud displacement vectors
- Visualise atmospheric movement patterns

## Methods

- Optical Flow
- Feature Tracking
- Image Segmentation
- Motion Vector Analysis

## Tools

Python  
OpenCV  
NumPy  
Matplotlib
